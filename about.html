The 5th Element

> Introduced in HTML5
> Used to draw graphics
> Through Javascript

Simple Filters

Create simple filters by extracting pixel data from the image, manipulating their values and then applying them to the canvas.

For example: "Invert" an image by subtracting each pixel's color value from 255, which is the max value.

Convolution Filters

Create convolution filters by extracting pixel data from the image and "adding some weight" to the surrounding pixels of each pixel.

For example: Blur
